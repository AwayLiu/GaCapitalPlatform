第五条 内部控制

5.1 内部控制七大循环

销售与收款

采购与付款循环

生产与仓储循环

货币资金循环

筹资与投资循环

固定资产和其他长期资产循环

工薪与人事循环

5.2 各循环关键控制点

5.2.1 货币资金循环

关键控制点

  1. 不相容职务分离：货币资金收付业务的出纳、审核与记录的职务相互分开；
  2. 所有收付款都有正式的审批手续；
  3. 所有收付款业务均及时正确入账。
  4. 银行存款账面余额与银行对账单余额调节相符。每月末进行一次与银行对账单的核对，编制银行余额调节表，经财务经理复核。
  5. 现金账面余额与实存现金相符：每月末由不负责现金的财务人员监盘库存现金，与现金日记账进行核对，若有差异查明原因，及时处理。
  6. 印章的安全使用-各分子公司参照集团公司的印章管理办法。

5.2.2 销售及收款循环

关键控制点：

  1. 仅接受在信用额度内的销售申请；
  2. 记录的销售均确已发出货物、销售货物交易均已于适当期间进行记录、已记录的销售退回均为真实发生的、已发生的销售退回均确已记录、已发生的销售退回均于恰当期间进行记录、准确计提坏账准备和核销坏账，并记录于恰当期间；
  3. 准确记录收款。收款均已记录且在恰当期间记录。监督应收账款及时收回；
  4. 顾客档案变更真实有效，均已于适当期间进行处理。顾客档案变更记录准确。确保顾客档案数据及时更新。

5.2.3 生产与仓储

内控关键控制点

  1. 已验收材料均附有有效采购订单、已验收材料均已准确记录、已验收材料均已记录于适当期间；
  2. 管理层授权进行生产；
  3. 发出材料均已准确记录、发出材料均记录于适当期间、已记录的生产成本均真实发生且与实际成本一致、已发生的生产成本均已记录、已发生的生产成本均记录于适当期间、存货流转均已准确地记录于适当期间、完工产品均已准确地记录于适当期间、产成品发运均已记录、已发运产成品均附有有效销售订单；
  4. 适当保管存货、准确记录存货价值、存货价值调整已于适当期间记录、存货价值调整是真实发生的。

5.2.4 采购与付款

内控关键控制点

  1. 采购计划安排合理、只有经过核准的采购订单才能发给供应商、已记录的采购订单内容准确、采购订单均已得到处理；
  2. 已记录的采购均确已收到物品、已记录的采购交易计价正确、采购物品交易记录于适当期间；
  3. 仅对已记录的应付账款办理支付，并准确记录付款、付款均已记录，且均于恰当期间进行记录；
  4. 对供应商档案的变更均为真实有效和准确的、供应商档案变更均已进行处理，且于适当期间进行处理、确保供应商档案数据及时更新。

5.2.5 长期资产

内控关键控制点

  1. 只有经核准的采购申请、合同才能执行；
  2. 新购入资产的验收；
  3. 固定资产的出入库及使用过程中迁移等均得到有效的控制和管理；
  4. 折旧费用、资产减值损失是真实的，均已准确记录，并记录于适当期间；
  5. 固定资产处置及转移均已准确记录

5.2.6 工薪与人事循环

关键控制点

  1. 员工名册新增项目均为真实有效的；新增员工均确已记入员工名册；离职员工均确已从员工名册中删除；员工名册删除项目均为真实有效的。
  2. 用以计算人员工资的工作时间数据均为实际工时；员工工作时间均已完整记录和输入；输入系统的时间记录均为准确的；
  3. 常备数据变动均为真实和准确的，并及时处理；只有经适当授权的人员才能接触工薪数据。
